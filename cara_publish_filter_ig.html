Cara Publish Filter Di Akun Instagram Anda

Haloo, disini kami akan memberi anda tutorial untuk melakukan publikasi filter pada akun instagram anda. Namun pertama anda harus menyiapkan beberapa syarat :

1. Akun Instagram harus terhubung dengan akun Facebook.

2. Siapkan Video Preview (MP4 atau MOV)

3. Filter yang akan dipublish (.arexport)

4. Icon Filter

File untuk diupload

Langsung saja kita mulai :

1. Login ke akun facebook anda
2. Buka halaman https://facebook.com/sparkarhub/

Anda akan diarahkan ke halaman dashboard Spark AR Hub

Halaman Dashboard Spark AR Hub

3. Pada pojok kanan atas tekan tombol Terbitkan Efek. Dan anda akan diarahkan ke halaman berikut
Form Publikasi Spark AR Hub

Ada beberapa form yang harus anda isi.

  • Nama : Isikan nama filter anda sesuai keinginan
  • Unggah File : Upload file .arexport yang telah saya berikan kepada anda
  • Platform : Sebaiknya pilih platform tertentu kemudian pilih berbagi pilih Instagram dan facebook. Seperti ini
  • Pemilik : Boleh dibiarkan seperti itu saja
  • Penerbit : Pilih akun Instagram untuk publish efek tersebut
  • Kategori : Pilih sesuai kategori, maksimal 4
  • Katakunci : Opsional, bisa dikosongkan
  • Video Demo : Upload video demo/preview yang telah disiapkan
  • Ikon : Upload Ikon yang sudah anda siapkan atau menggunakan ikon dari video demo
  • Tanggal Penerbitan : Pilih segera

4. Terakhir klik tombol ajukan
5. Selesai – Efek anda sedang ditinjau, biasanya dalam 1-5 hari
